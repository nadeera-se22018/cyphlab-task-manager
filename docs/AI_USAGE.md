# AI Usage Report

In the development of the Cyphlab Task Manager, AI tools were leveraged to accelerate implementation and structure design patterns while maintaining full human design control.

## Tools Used

### Gemini
- Used for initial system architecture planning.
- Assisted in PostgreSQL database schema layout design and Prisma relationship setup.
- Provided step-by-step design guidance.

### Antigravity AI IDE
- Assisted in writing the boilerplate structure for the Express.js server and database client integration.
- Facilitated creation of responsive React views using Tailwind CSS.
- Helped script modular Express.js route handlers.
